# Hola bienvenido a mi "Alke Wallet".

Este es el proyecto final del modulo 2.

## Descripción 
Esta es una billetera virtual que nos permite realizar tanto transferencias como depositos.

## Funcionaliad y tecnologías

En esta Ewallet su estructura principal se baso en html, JavaScript, Css y Jquery, tambien se implemento el uso de "Bootstrap" como framework.
En sus funcionalidades  quedo implementado  un **Login** como validacion y creación de usuario , un **Saldo** para una visualización simple del monto disponible
y un apartado de **Movimientos** Con la implementacion de un historial de depósitos y transferencias .
